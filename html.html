What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML consists of a series of elements.
  • HTML elements tell the browser how to display the content.
  • HTML is the language for describing the structure of Web pages.

    Create a Simple Webpage

    Step 1: Create a new folder
    Step 2: Open Visual Studio Code
    Step 3: Select the created folder
    Step 4: Create a new file called index.html
    Step 5: Type the following code

              
            <html>
              <head>
                <title>First Webpage</title>
              </head>
              <body>
                <h1>This is my first webpage.</h1>
              </body>
            </html>
    Step 6: Save the file
    Shortcut Key Ctrl + s
    Step 7: Open the created folder and click on the index.html file.
    You will be able to see your Webpage.
